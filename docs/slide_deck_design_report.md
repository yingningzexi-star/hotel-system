## 如何导出为高保真 PDF（供上传使用）

为了符合学校上传 PDF/图片的要求，我特别为您制作了专用于打印导出的文件：

1. **打开打印预览页面**：
   在本地服务启动状态下，使用 Chrome 或 Edge 浏览器访问：
   👉 [http://localhost:4311/hotel-system-deck/deck-print.html](http://localhost:4311/hotel-system-deck/deck-print.html)
   *(打开此页面后，系统会自动检测资源加载并为您弹出浏览器打印对话框)*。

2. **配置打印参数**：
   在打印对话框中，请务必进行以下设置，以确保 PDF 的比例和样式完全一致：
   * **目标打印机**：选择 **“另存为 PDF”** (Save as PDF)。
   * **布局**：选择 **“横向”** (Landscape)。
   * **纸张大小**：选择 **A4** (或者在“更多设置”中选择“无边距”以达到 16:9 最佳填充)。
   * **页边距**：选择 **“无”** (None)。
   * **选项**：必须**勾选“背景图形”** (Background graphics)，否则背景色和渐变将变为白色。
   * **选项**：**取消勾选“页眉和页脚”** (Headers and footers)，防止边缘出现网址和日期。

3. **保存**：
   点击 **“保存”** 并选择保存路径，您就可以获得一份无缩放、无白边的高清高保真答辩 PDF！
