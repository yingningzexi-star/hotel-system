# 开发用户登录、注册与权限拦截系统

本计划旨在实现组员 A 负责的“用户、权限和信用管理”模块。这包括用户数据库实体的 JPA 映射、BCrypt 密码加密、服务逻辑、安全登录拦截器、相关控制器以及配套的 Thymeleaf 视图页面。

## User Review Required

> [!IMPORTANT]
> 1. **密码加密依赖**：在 `pom.xml` 中引入了 `spring-security-crypto` 依赖。我们并未使用全套 Spring Security 过滤器链（因为其默认拦截行为较复杂，不适合本阶段轻量级开发），而是使用其 `BCryptPasswordEncoder` 进行密码哈希，辅以 Spring MVC 拦截器实现轻量级登录拦截。
> 2. **信用控制**：目前规则是“累计取消 3 次，禁订 7 天”。在登录和检查时，系统会自动对比当前时间与 `banned_until`。如果已过禁订期，将自动解除禁订并清空计数器。
